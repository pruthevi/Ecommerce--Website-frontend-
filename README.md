Ecommerce-Website-Frontend::

A clean and simple ecommerce website user interface built using pure HTML and CSS. This project displays a set of products with images, descriptions, and pricing inside a structured card layout. The design includes a top navigation bar, product grid section, hover-effects, buttons, and a minimal UI suitable for beginners learning web development. The goal of this project is to create a basic ecommerce-style layout focusing on clean design, simple styling, and easy code structure while maintaining a responsive and user-friendly interface.
